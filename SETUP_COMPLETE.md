## 🚀 Quick Start Commands

### Create a New Project
```powershell

```

### Update Project Status
```powershell
.\_scripts\update-status.ps1
```
Or directly:
```powershell
.\_scripts\update-status.ps1 -ProjectName "TorchinDesiree" -Status "production" -Note "v1.0 released"
```

### View Project Dashboard
