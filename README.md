The Funky Black Box is the master stash for anything that keeps our development world spinning.
It’s part emergency kit, part daily driver toolbox, and all groove.

🛠 What Goes In Here?
Emergency Scripts – recovery/diagnostics for “oh crap” moments

Everyday Tools – scripts, aliases, and commands we use constantly

Config Files – env setups, .env.example templates, protection rules

Preflight Rituals – steps to run before installing/upgrading anything

Special Assets – things so valuable we’d cry if they vanished


📂 Suggested Branches
main – core resources and must-haves

emergency-scripts – recovery & protection focused

dev-helpers – everyday speed-up tools


🎯 Mission
Keep the torch lit. Protect the flow.
Crash landing or smooth launch, The Funky Black Box gets you back in the air — with style.

⚠ Rules of the Funky Black Box
Never delete from here without a backup.

Keep things organized and labeled — future you will thank you.

If it saves your butt, it belongs in here.


🖥 Script Index
A quick guide to the scripts stored in this box.
system-check.sh   — Runs a full health check on the dev environment      (/emergency-scripts)
env-setup.sh      — Sets up environment variables for new installs       (/dev-helpers)
backup-vault.sh   — Creates a compressed backup of key project files     (/emergency-scripts)
script-name.ext   — Short description of what it does                    (/folder-name)
